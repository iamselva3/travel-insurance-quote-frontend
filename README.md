# Travel Insurance

A responsive travel insurance interface built using pure HTML, CSS, and JavaScript based on the provided Figma design.

## Live Links
- **Prototype:** https://travel-insurance-quote-frontend.vercel.app/

---

## Pages Included
1. Landing Page
2. Quote Form
3. Plan Selection Page

---

## Tech Stack
- HTML5
- CSS3
- JavaScript
- Flatpickr (date picker)

No frameworks used.

---

## Features
- Fully responsive layout
- Mega navigation menu (desktop)
- Mobile slide-out navigation
- Multi-destination selector
- Traveller age inputs
- LocalStorage form persistence
- Mobile accordion for plans
